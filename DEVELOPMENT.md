# Instrucciones de desarrollo

## Requisitos previos
- Flutter SDK (3.19.0 o superior)
- Android Studio con el SDK de Android
- (Opcional) Visual Studio Code con extensiones Flutter y Dart
- Firebase CLI
- Node.js y npm
- Git

## Configuración del entorno
1. Clona el repositorio
2. Ejecuta \lutter pub get\ en cada aplicación y paquete
3. Configura Firebase con \lutterfire configure\

## Estructura del proyecto
Ver README.md para la estructura general del proyecto.

## Comandos útiles
- \lutter run\ - Ejecutar la aplicación en modo debug
- \lutter build apk\ - Compilar APK para Android
- \lutter build ios\ - Compilar para iOS
- \lutter build web\ - Compilar para web
- \lutter test\ - Ejecutar pruebas
