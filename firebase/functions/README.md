# GoPlus Firebase Cloud Functions

Backend serverless para la plataforma GoPlus.

## Estructura

- **auth/**: Funciones relacionadas con autenticación
- **orders/**: Funciones para gestión de pedidos
- **notifications/**: Funciones para envío de notificaciones
- **payments/**: Funciones para procesamiento de pagos

## Desarrollo local

1. Instalar dependencias:
npm install

2. Ejecutar emuladores:
npm run serve

## Despliegue
npm run deploy
